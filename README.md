# Guías
Este repositorio contiene una colección de guías, plantillas y rúbricas desarrolladas para estandarizar y elevar la calidad de la documentación técnica y la comunicación científica. 

Este conjunto de recursos se centra en dos pilares fundamentales:
- Formato IEEE: Promover el uso correcto del estilo de citación numérica, la nomenclatura y la estructura formal de la IEEE en documentos técnicos.
- Comunicación Efectiva: Enseñar a los estudiantes a comunicar resultados complejos de ingeniería de forma concisa, clara y visualmente atractiva, tanto en documentos extensos como en formatos de presentación rápida como el póster científico.
